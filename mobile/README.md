# Blur Mobile - React Native App

A production-grade React Native mobile application for Blur, an anonymous chat platform with real-time messaging, push notifications, and beautiful animations.

## 🎯 Features

- **Authentication System**: Secure login and registration with persistent sessions
- **Real-time Chat**: Socket.IO integration for instant messaging across web and mobile
- **Push Notifications**: Room-specific notification preferences with Expo Notifications
- **Message Reactions**: Quick reactions with emoji support
- **Message Replies**: Thread conversations with reply functionality
- **Beautiful UI/UX**: Custom animations, haptics, and micro-interactions
- **Dark Theme**: Distinctive design with Manrope font and pink/red accent colors
- **Cross-Platform**: Works seamlessly with the web frontend

## 🛠 Tech Stack

- **Framework**: React Native with Expo SDK 54
- **Language**: TypeScript
- **Navigation**: Expo Router (file-based routing)
- **Real-time**: Socket.IO Client
- **Animations**: React Native Reanimated 4
- **Icons**: Lucide React Native
- **Fonts**: Manrope (Google Fonts)
- **Notifications**: Expo Notifications
- **Storage**: AsyncStorage
- **Haptics**: Expo Haptics
